Analytics with GeoMesa
======================

GeoMesa provides several resources for analytics on spatio-temporal data.

 * :doc:`spark` allows execution of jobs on Apache Spark using data stored in GeoMesa Accumulo or any other GeoTools-backed data store.
 * :doc:`process` describes a set of analytic WPS processes for use in GeoServer for heatmap generation, KNN searches, statistical sampling, and more.
 * :doc:`stream` provides tools to process streams of ``SimpleFeature``\ s.
