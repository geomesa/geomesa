HBase Data Store
================

The GeoMesa HBase Data Store is an implementation of the GeoTools
``DataStore`` interface that is backed by `Apache HBase`_.
It is found in the ``geomesa-hbase`` directory of the GeoMesa
source distribution.

.. _Apache HBase: https://hbase.apache.org/

To get started with the HBase Data Store, try the :doc:`/tutorials/geomesa-quickstart-hbase` tutorial.

For instructions on how to install and configure the HBase Data Store, including the command line tools, see :doc:`/user/installation/hbase`.
